1. ONBOARDING PHILOSOPHY
================================================================================

First impressions matter. The setup wizard must:
- Feel welcoming, not intimidating
- Respect user's time
- Explain privacy choices clearly
- Never use dark patterns
- Work for ages 8-80

4. EXPRESS SETUP
================================================================================

For experienced users:

TRIGGER: Tap "Express Setup" on welcome screen

EXPRESS FLOW:
1. Language
2. WiFi
3. PIN
4. Biometric
5. Done

DEFAULTS APPLIED:
- Recommended privacy settings
- Standard mode
- No account
- No data migration

================================================================================
5. RECOVERY SETUP
================================================================================

After factory reset or restoration:

OPTIONS:
• Restore from backup
• Set up as new
• Import from Android

RESTORE FLOW:
1. Select backup location (local/cloud)
2. Authenticate
3. Select what to restore
4. Apply
5. Brief re-setup (PIN, biometric)

================================================================================
8. SETUP ANALYTICS
================================================================================

WHAT WE DON'T DO:
- Track setup completion
- Record user choices
- Send any data to servers

WHAT WE COULD DO (opt-in only):
- Anonymous aggregate statistics
- Setup failure reporting
- Only if user explicitly opts in
- Disabled by default

================================================================================
9. ACCESSIBILITY THROUGHOUT SETUP
================================================================================

REQUIREMENTS:
- All screens work with screen reader
- All text scalable
- All colors meet contrast requirements
- All actions have text alternatives
- Setup works without touch (physical buttons)
