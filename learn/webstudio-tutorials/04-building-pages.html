Understanding Layouts

A layout is how you arrange things on your page. Most websites have these parts:

Header (Logo & Navigation)
Sidebar
Main Content

This is where your important stuff goes!

Footer (Contact info, links)
Common Layouts:
  • Header + Content + Footer (most simple)
  • Header + Sidebar + Content + Footer (for blogs)
  • Full-width sections (for landing pages)
2

Using Flexbox for Layouts

Flexbox is a super powerful way to arrange things! It makes boxes line up nicely.

How to use Flexbox:

  1. Add a Box element to your page
  2. Select it and go to the Style Panel
  3. Find "Display" and change it to "Flex"
  4. Now things inside will line up in a row!
Flexbox Settings:
  • Direction: Row (side by side) or Column (stacked)
  • Justify: How to spread things out horizontally
  • Align: How to position things vertically
  • Gap: Space between items

Adding More Pages

Websites often have multiple pages! Like Home, About, Contact, etc.

Home
About
Gallery
Contact

How to add a new page:

  1. Look for the Pages panel (usually on the left)
  2. Click the "+" or "New Page" button
  3. Give your page a name (like "about" or "contact")
  4. Start building on your new page!
Linking Pages: Use Link elements or Buttons to connect your pages! Set the link to the page path (like "/about" or "/contact").
4

Creating a Navigation Menu

A navigation menu helps visitors find their way around your website!

How to build a nav menu:

  1. Add a Box element at the top of your page
  2. Set it to display: flex with direction: row
  3. Add Link elements inside for each page
  4. Style them to look like buttons or tabs
  5. Copy this to all your pages!
Pro Tip: In Web Studio, you can create "components" that you reuse across pages. Your nav menu is perfect for this!
5

Preview and Test

Before showing your website to the world, always test it!

How to preview:

  1. Click the "Preview" button (usually at the top)
  2. Your website opens in a new tab
  3. Test all your links and buttons
  4. Try it on different screen sizes (phone, tablet, computer)
Test Checklist:
  • Do all links work?
  • Can you read all the text?
  • Do images load properly?
  • Does it look good on your phone?
6

Final Project: Build Your Own Website!

Now it's time to put everything together! Create a complete website with these pages:

  1. Home Page - Welcome message, what your site is about
  2. About Page - Tell visitors about yourself or your topic
  3. Gallery Page - Show pictures of things you like
  4. Contact Page - How people can reach you

Remember to include:

  • A navigation menu on every page
  • Consistent colors and style
  • At least one image per page
  • Working links between pages
